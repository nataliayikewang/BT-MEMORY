\subsubsection{Project Starting}

A solid project start requires a preliminary study and initial project definition. This aligns the project vision among all involved members and establishes a unified understanding of the project's scope and objectives. To achieve this, the following tasks are defined:
 
\textbf{PS1. Context and Scope}

This tasks covers the following sub-tasks:
\begin{itemize}[noitemsep]
    \item Problem and stakeholder identification
    \item Proposed solution
    \item Context and market study
    \item Scope definition (objectives, requirements, obstacles, and risks)
    \item Development methodology selection
    \item Monitoring and validation tools selection
\end{itemize}

\textbf{PS2. Temporal Planning}

Identify all tasks and resources, design a Gantt diagram, and define a risk management strategy.

\textbf{PS3. Budget and Sustainability}

Define a project budget and analyze the sustainability aspects of the proposed solution.

\textbf{PS4. Initial Documentation}

Consolidate the documentation from previous tasks into the definitive version of the initial documentation.
