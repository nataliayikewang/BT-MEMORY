\subsubsection{Project Starting}

Before delving into the practical aspects of the project, a preliminary 
study and initial project definition are required. This 
aligns the project vision among all involved members and establishes a 
unified understanding of the project's scope and objectives. To fully 
comprehend the entanglement of each task, a list of entailed sub-tasks 
or a brief definition is provided below each task.

\textbf{PS1. Context and Scope}

\begin{itemize}
    \item Problem and stakeholder identification
    \item Proposed solution
    \item Context and market study
    \item Scope definition (objectives, requirements, obstacles, and risks)
    \item Development methodology selection
    \item Monitoring and validation tools selection
\end{itemize}
\newpage

\textbf{PS2. Temporal Planning}

\begin{itemize}
    \item Tasks and resources identification
    \item Gantt diagram
    \item Risks management
\end{itemize}

\textbf{PS3. Budget and Sustainability}

\begin{itemize}
    \item Project budget calculation
    \item Sustainability analysis
\end{itemize}

\textbf{PS4. Initial Documentation}

Consolidate the documentation from previous tasks into the definitive version of the initial documentation.
