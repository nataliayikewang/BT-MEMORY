\subsubsection{Project Management}

As the name suggests, this section includes all tasks  related to project 
management. Some tasks are split into smaller sub-tasks, while others 
include a brief description.

\textbf{PM1. Agile Ceremonies}

\begin{itemize}
    \item Scrum daily meetings
    \item Sprint planning
    \item Sprint review
    \item Sprint retrospective
\end{itemize}

\textbf{PM2. Thesis Director Review Meetings}

Meet with the thesis director, allowing him to catch up and monitor 
project progress.

\textbf{PM3. Git Repository Documentation}

\begin{itemize}
    \item Application setup guide
    \item Repository documentation
    \item Code specification comments
\end{itemize}

\textbf{PM4. User Manual Documentation}

Write a user manual that serves as a reference guide for the application's
 end-users.

\textbf{PM5. Product Backlog Update}

Refine, update, and maintain the product backlog, ensuring it always 
reflects the current project status.

\newpage

\textbf{PM6. Final Thesis Documentation}

Write the bachelor's thesis final documentation.

\textbf{PM7. Final Demonstration Video}

Record a final demonstration of the application once it is ready for use.


\textbf{PM8. Bachelor's Thesis Defense}

Plan and prepare a final exposition to defend and justify the bachelor's 
thesis achievements, while demonstrating commitment to the academic 
standards.
