\subsubsection{Project Management}

As the name suggests, this section includes all tasks  related to project management. 

\textbf{PM1. Agile Ceremonies}

Run scrum daily meetings, sprint plannings, sprint reviews, and sprint retrospectives. 

\textbf{PM2. Thesis Director Review Meetings}

Meet with the thesis director, allowing him to catch up and monitor project progress. 

\textbf{PM3. Git Repository Documentation}

Maintain clear repository documentation, including an application setup guide and meaningful code comments.

\textbf{PM4. User Manual Documentation}

Write a user manual that serves as a reference guide for the application's end-users.

\textbf{PM5. Product Backlog Update}

Refine, update, and maintain the product backlog, ensuring it always reflects the current project status.

\textbf{PM6. Final Thesis Documentation}

Write the bachelor's thesis final documentation.

\textbf{PM7. Final Demonstration Video}

Record a final demonstration of the application once it is ready for use.


\textbf{PM8. Bachelor's Thesis Defense}

Plan and prepare a final exposition to defend and justify the bachelor's thesis achievements, while demonstrating commitment to the academic standards.
