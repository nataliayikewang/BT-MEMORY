\subsection{Resources}

Following the analysis of all required tasks for successful project 
completion, all necessary resources must be identified.  Resources 
are divided into two main groups: human resources and physical resources. 
All human resources are named in Table \ref{table:human-resources-table}, specifying the responsible 
individual for each role. The physical resources are further categorized 
into software and hardware resources, and are identified and briefly 
described in Table \ref{table:hardware-resources-table} and \ref{table:software-resources-table}.
